# 🔐 Password Generator

A modern and responsive Password Generator website built using HTML, CSS, and JavaScript.  
This is my first frontend practice project to improve my web development skills.

---

## 🚀 Features

- Generate secure random passwords
- Select password length
- Include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Copy password to clipboard
- Responsive modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
